# Tips for Competitive Programming Challenges



### C/C++ 환경을 visual studio에서 세팅하기

- `bits/stdc++.h` 헤더에는 흔히 사용하는 헤더들이 대부분 포함되어 있습니다. 그렇기 때문에, visual studio에서도 세팅을 하면 좋습니다.
- [stdc++.h 파일](stdc++.h)을 다운로드 받아서 `C:\Program Files\Microsoft Visual Studio\2022\Community\VC\Tools\MSVC\14.30.30705\include\bits` 폴더에 넣습니다. bits 폴더는 존재하지 않기 때문에 폴더 생성후 넣으면 됩니다.
